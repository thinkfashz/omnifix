# Omnifix - variables .env para Vercel

Copia estas variables en Vercel > Project Settings > Environment Variables.
No pegues secretos reales en GitHub. Usa valores propios en Vercel.

## Base pública

NEXT_PUBLIC_WHATSAPP_NUMBER=56930121625

## Cómo generar secretos

En una terminal puedes usar:

openssl rand -base64 48

Genera valores distintos para:
- ADMIN_SESSION_SECRET
- ADMIN_PASSWORD_PEPPER
- ADMIN_INIT_SECRET

Después del primer login/admin init, puedes quitar ADMIN_INITIAL_PASSWORD y ADMIN_INIT_SECRET de Vercel para cerrar la inicialización.
