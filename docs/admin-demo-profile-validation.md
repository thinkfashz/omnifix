# Validacion demo y perfil admin

Checklist de validacion para el flujo demo 24h y el perfil administrativo.

## Rutas principales

- /admin/equipo/demo
- /admin/perfil

## Validar demo

- Crear link demo 24h.
- Abrir link demo en otro navegador.
- Confirmar modo viewer.
- Confirmar bloqueo por IP.
- Confirmar registro de accesos.
- Enviar feedback demo.

## Validar perfil

- Abrir /admin/perfil.
- Editar nombre, telefono, bio y redes.
- Guardar cambios.
- Subir avatar desde /admin/perfil.
- Confirmar card visual de perfil.
- Confirmar acceso a Perfil desde menu hamburguesa.
- Confirmar acceso a Perfil desde navbar y sidebar.
- Confirmar acceso a Links demo 24h desde menu hamburguesa y sidebar.
