# Presupuestos Vercel Preview

Este archivo documenta el deploy preview del módulo de presupuestos.

Rutas principales para probar:

- `/admin/presupuestos`
- `/presupuestos/trima-mobiliario-modular-laboratorio-container`

Flujo recomendado:

1. Crear o editar presupuesto.
2. Guardar y registrar en base de datos.
3. Copiar link público.
4. Probar vista móvil.
5. Probar confirmación por WhatsApp.
6. Probar carga de imágenes y archivos técnicos.

Este cambio existe para generar un nuevo Pull Request y disparar un deploy preview limpio en Vercel.
